# Nelly 🐶

Dieses Repository enthält die **Grunddaten und das Gedächtnis**
für einen KI‑Agenten rund um unseren Hund Nelly.

## Zentrale Identität
- `data/identities/nelly.yaml`

Diese Datei beschreibt **wer Nelly ist**.
Alle KI‑Agenten müssen sie beim Start laden.

## Grundregel
Wenn von „Nelly“, „sie“ oder „dem Hund“ gesprochen wird,
ist immer diese Identität gemeint.
